## SweetCrumbs Bakery Website

SweetCrumbs Bakery is a static HTML website created for an artisan bakery located in Nairobi, Kenya. The website showcases 
the bakery's products, weekly specials, customer testimonials, gallery, business information, and a contact/order inquiry form.

## Features

* Multi-section navigation menu
* Bakery introduction and company story
* Product menu with pricing and availability
* Weekly promotional offers
* Customer testimonials
* Bakery showcase gallery
* Contact and inquiry form
* Business hours and location information
* Traditional HTML table-based layout
* Single-page website structure

## Technologies Used

* HTML5
* HTML tables for layout
* HTML presentation attributes
* Native HTML form elements
* Anchor-based navigation

## Purpose

This project demonstrates the creation of a complete business website using only HTML tables, attributes, and native form elements, 
without CSS, JavaScript or external frameworks.

## Project Structure

index.html

A single, self-contained HTML file containing all website sections and functionality.

## Learning Objectives

* Building web pages using pure HTML
* Creating table-based layouts
* Working with forms and user input elements
* Implementing internal page navigation with anchors
* Structuring a complete business website without CSS or JavaScript
